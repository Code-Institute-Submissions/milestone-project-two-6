How to Play

Desktop Mode

  1. Control the snake using the control keys on your keyboard
  2. Avoid hitting your tail or the walls
  3. Score is shown on top of the screen
  4. Final Score is shown when player loses

Smartphone/Tablet Mode

  1. Control using touch controls on the screen/li>
  2. Remaining rules are same as above
